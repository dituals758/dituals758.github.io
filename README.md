# WeekFlow — Минималистичные Часы

Профессиональные минималистичные часы с цветовой кодировкой дней недели.

## Демо
[https://dituals758.github.io](https://dituals758.github.io)

## Установка
1. Откройте [https://dituals758.github.io](https://dituals758.github.io)
2. Нажмите "Установить" в правом верхнем углу
3. Добавьте приложение на рабочий стол

## Особенности
- Четкая типографика с моноширинным шрифтом
- Визуализация прогресса текущего дня
- Уникальный цвет для каждого дня недели
- Полностью адаптивный дизайн
- Мгновенная загрузка (PWA)
- Работает полностью офлайн

## Цвета дней недели
- Понедельник: #3a86ff
- Вторник: #38b6ff
- Среда: #8338ec
- Четверг: #ff006e
- Пятница: #fb5607
- Суббота: #06d6a0
- Воскресенье: #ef476f

## Технологии
- HTML5, CSS3, Vanilla JavaScript
- Progressive Web App (PWA)
- Service Worker
- CSS Custom Properties
- Responsive Design

## Файлы проекта
- `index.html` - главная страница
- `sw.js` - Service Worker для офлайн работы
- `manifest.json` - конфигурация PWA
- `favicon.ico` - иконка сайта
- `icon-*.png` - иконки для PWA
- `404.html` - страница 404
- `robots.txt` - правила для поисковых систем
- `sitemap.xml` - карта сайта
- `.nojekyll` - отключает Jekyll на GitHub Pages
- `README.md` - документация
